## 🦄 Unicode 特殊字符

### 大小写转换魔法

| 小写字符 | Unicode | toUpperCase() 结果 | 用途 |
|----------|---------|-------------------|------|
| `ſ` | U+017F | `S` | 绕过 `[a-z]` 过滤 |
| `ı` | U+0131 | `I` | 绕过 `[a-z]` 过滤 |
| `ﬁ` | U+FB01 | `FI` | 一个字符变两个 |
| `ﬂ` | U+FB02 | `FL` | 一个字符变两个 |

### HTML 实体漏分号吞并

当 HTML 实体缺少分号 `;` 时，浏览器会贪婪匹配后面的十六进制字符：

| 输入 | 错误实体 | 被解析为 | Unicode |
|------|----------|----------|---------|
| `/a` | `&#x2fa` | `˺` | U+02FA |
| `/b` | `&#x2fb` | `˻` | U+02FB |
| `/c` | `&#x2fc` | `˼` | U+02FC |
| `/d` | `&#x2fd` | `˽` | U+02FD |
| `/e` | `&#x2fe` | `˾` | U+02FE |
| `/f` | `&#x2ff` | `˿` | U+02FF |
| `/0` | `&#x2f0` | `˰` | U+02F0 |

**安全字符**（不会被吞并）：`g-z`、`G-Z`

## 💡 编码技巧

### 1. 混合编码
```html
<!-- HTML + URL 混合 -->
<a href="&#106;avascript:%61lert(1)">click</a>
```

### 2. 多重编码
```
原文: <
URL: %3C
双重URL: %253C
```

### 3. 大小写混淆
```html
&#X3C;  <!-- 十六进制 X 大写也有效 -->
&#x3C;  <!-- 标准写法 -->
```
